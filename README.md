# Raid Bet Master

**插件名称**：Raid Bet Master  
**适用游戏**：World of Warcraft Classic (Classic 泰坦重铸服)  
**版本**：1.0.0  
**作者**：jerry  

---

## 插件简介

Raid Bet Master 是一个专为副本 GDKP / 下注活动设计的 WoW Classic 插件。  

它可以帮助副本庄家：

- **记录玩家下注**（聊天贴装备 + 交易金币）  
- **管理下注倒计时 & 锁定机制**  
- **统计副本内金币变化**（支持识别掉落和卖装备收入）  
- **Boss掉落自动赔付**（根据下注赔率计算）  
- **保存历史记录**，可手动清理或删除  
- 提供 **WeakAuras级 UI**，直观显示下注、交易和统计信息  

> 插件仅记录游戏内事件，不会自动执行交易或修改游戏数据。

---

## 功能列表

1. **下注记录**  
   - 玩家交易金币 → 插件记录下注金额和玩家角色名  
   - 聊天贴装备 → 自动匹配到下注装备  
   - 支持手动调整或分配错误的下注  

2. **倒计时与锁定**  
   - 可设置下注时间  
   - 倒计时结束后自动锁定，不允许新增下注，只能修改已下注项  

3. **收益统计**  
   - 自动统计下注收入、赔付、掉落金币、卖装备收益  
   - 显示副本开始和结束金币，支持未知来源金币标记和删除  

4. **历史记录管理**  
   - 自动保存最近 N 次副本记录（默认 10 次）  
   - 可清理历史记录  

5. **用户界面**  
   - 简单的风格界面  
   - 主界面展示下注列表、交易记录、收益统计  
   - 可拖动窗口，自定义布局  

---

## 安装方法

1. 下载插件 ZIP 文件并解压  
2. 将 `RaidBetMaster` 文件夹放入你的 `Interface/AddOns/` 目录  
3. 登录游戏，在角色选择界面确保插件已启用  
4. 游戏内输入 `/rbm` 打开插件主界面  

---

## 配置

- 默认下注金额：100 金币  
- 默认赔率：1:10  
- 默认下注倒计时：30 秒  
- 历史记录保存数量：10  

> 以上参数可在插件 UI 或 Config.lua 文件中修改  

---

## 使用说明

1. **开始副本**  
   - 插件会记录当前包裹金币作为起始金币  

2. **玩家下注**  
   - 玩家交易金币并贴出装备  
   - 插件自动记录交易金额和装备匹配  

3. **Boss掉落结算**  
   - 插件计算每件装备的返还金额  
   - 若掉落未下注装备 → 通杀（庄家保留金币）  
   - 支持手动标记已结算  

4. **历史记录与统计**  
   - 副本结束后自动保存统计  
   - 可通过 UI 查看历史记录或清理
